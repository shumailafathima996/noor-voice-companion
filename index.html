Judge clarification: Offline guidance, access & security

Noor is offline-first, not a live Google Maps replacement. Offline guidance works for saved places only. A teacher, parent, or guardian can save routes once, such as classroom, lab, library, home, or washroom.

1. Saved-place guidanceNoor reads step-by-step directions already saved on this device. Example: “Guide me to library.”
2. New place fallbackIf the place is not saved, Noor says it cannot guide offline and asks the user to connect internet, share location, or call family/helper.
3. App accessUsers can open Noor in a browser link. After first loading, core voice actions work on the same device without API keys.
4. Contact securityFamily contacts, medicine notes, and saved routes stay only in this device storage. Noor does not upload them to any server/cloud.

Saved place demo: User says “Hey Friend, guide me to library.” Noor replies with the saved route.

Unknown place demo: User says “Hey Friend, guide me to auditorium.” Noor replies: “Auditorium is not saved offline. Please connect internet, share location, or call family for help.”

Permission demo: Mic, location, calling, and SMS are controlled by browser/phone permission. Noor never silently accesses contacts.
